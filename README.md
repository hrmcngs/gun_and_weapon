# gun_and_weapon
TaCZ addon - Gunblade mod for Minecraft Forge 1.20.1

## Build

```bash
JAVA_HOME="C:/Program Files/Java/jdk-17" ./gradlew build
```

JAR is output to `build/libs/gun_and_weapon-1.0.jar`

## Install

1. `build/libs/gun_and_weapon-1.0.jar` to `.minecraft/mods/`
2. `tacz-1.20.1-1.1.7-hotfix.jar` to `.minecraft/mods/`
3. `.minecraft/tacz/gunblade_pack/` exists, delete it to force reinstall
